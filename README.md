# Superstore-Sales-Analysis
Sales analysis project using Python, MySQL and Power BI

# Tools Used:
* Python: Data cleaning aur exploration.
* MySQL: SQL queries for sales and profit insights.
* Power BI: Interactive dashboard for regional analysis.

# Key Findings:
* Identification of top-performing product categories.
* Profitability analysis across different regions.
* Yearly sales growth trends.

# Files:
* analysis.py (Cleaning Code)
* superstore_project_queries.sql (SQL Code)
* Superstore Sales Dashboard.pbix (Power BI File)
